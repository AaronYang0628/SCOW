# 部署认证系统

认证系统(`auth`)为系统提供用户登录功能，以及为各个组件提供鉴权服务。

# 内置认证系统

我们提供的认证系统实现支持以下认证方式。请根据自己的集群的情况，在选择自己的认证方式并进行部署。
  
- [SSH](../auth/ssh.md)
- [LDAP](../auth/ldap.md)

:::tip

如果您的集群可以使用SSH登录，可以首先使用最简单的SSH方式进行测试。

:::

# 自定义认证方式

如果您的集群需要使用其他认证方式，您可以参考[自定义](../auth/custom.md)文档自己实现一个认证系统。
